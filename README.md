<h1><b>SHADOWPURGE</b></h1>

**A clean, fast, and powerful opera minipay eraser.**

**ShadowPurge** is a modern CLI client designed for smooth and efficient operations with a clean, and responsive interface.

---

## Features
- Clean and easy-to-use interface  
- Support for single file, series by prefix, or all cookies in folder  
- Built-in bulk cookie creator  
- Local file management (delete cookies)  
- Secure API key authentication with auto-save  
- Optimized for Termux and desktop environments  

---

## Installation

### Termux
```bash
pkg update && pkg upgrade -y
pkg install git python -y
pip install httpx

git clone --depth=1 https://github.com/ultra000x/shadowpurge.git
cd shadowpurge
git pull
python client.py
```

### Windows / Linux / macOS
```bash
git clone --depth=1 https://github.com/ultra000x/shadowpurge.git
pip install httpx
cd shadowpurge
git pull
python client.py
```

---

### Requirements
- Python 3
- `httpx` library
- Termux (recommended for mobile)

### Privacy & Data Security
- All operations are performed locally on your device.
- Cookie files are never stored on the server.
- The client only sends data when you explicitly start a job.
- No logs, cookies, or personal data are collected by the tool.
